\# FLACA Best Practice - Tutorial



\## Introduction



The challenge of using Large Language Models (LLMs) for the automatic detection of concepts such as stances and frames in real-world cases of media coverage often lies in the fact that classification accuracy is constrained by the ambiguity and complexity of underlying concepts, as well as class imbalances in media coverage. 

This paper aims to address these challenges by comparing approaches of fine-tuning BERT-like transformer models versus few-shot classification through LLMs for the purpose of classifying a specific media case. Specifically, it poses the question of how the performance of fine-tuning a classifier using approx. 100-200 examples compares to the outputs of different LLM prompts (using different GPT and smaller open source models).
